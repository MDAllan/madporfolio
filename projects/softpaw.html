SoftPaw Cleaning Company

SoftPaw is a modern cleaning service platform designed to streamline how clients book home or commercial cleaning services. From a simple and elegant user dashboard to a powerful admin panel, SoftPaw focuses on user experience, seamless booking, and efficient backend management.

Project Overview

SoftPaw Cleaning is a full-stack web application built as part of a capstone project. The goal was to design a complete booking platform where users can schedule cleanings, and administrators can manage services, bookings, and providers from a centralized dashboard.

My Responsibilities

  • Developed the full front-end using HTML, CSS, and JavaScript with a focus on responsive layout and user-friendly design.
  • Built a functional back-end using PHP and MySQL to handle service listings, user registrations, bookings, and reviews.
  • Created a secure authentication system with hashed passwords using PHP's `password_hash()` and `password_verify()` functions.
  • Designed and implemented dynamic dashboards for both users and administrators, featuring filters, live booking updates, and service management tools.
  • Handled the entire database schema, covering users, service providers, bookings, reviews, and services.
  • Used JavaScript for client-side form validation, dashboard interactions, and conditional logic for booking options.

Visual identity

Key Features

  • User-friendly booking form with smart conditional logic.
  • Service provider dashboard for adding, editing, and deleting services with image support.
  • Admin panel with booking and user filtering, provider approvals, and user management tools.
  • Integrated review system linked to service and provider performance.
  • Responsive layout for mobile and desktop users.

Security & Performance

Security was a top priority throughout the project. All user passwords are encrypted using PHP's built-in password hashing methods, and user input is sanitized to prevent SQL injection and XSS attacks. Booking and service data is loaded using efficient SQL queries to maintain speed and reliability across all devices.

Final Deliverables

  • Responsive front-end website with clean design and interactive UI elements.
  • Admin and provider dashboards with dynamic features for service management and user control.
  • Complete database with all necessary relational tables (users, providers, bookings, reviews, services).
  • Secure user login/register functionality with password encryption.
  • JavaScript enhancements for interactivity and real-time form updates.
